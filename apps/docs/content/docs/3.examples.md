# Examples

## Basic variants

```ts
sileo.success({ title: "Changes saved" })
sileo.error({ title: "Upload failed", description: "File exceeds 10 MB." })
sileo.warning({ title: "Storage at 90%", description: "Archive old projects." })
sileo.info({ title: "v2.4 available", description: "Dark mode + custom positions." })
sileo.loading({ title: "Syncing workspace…" })
sileo.show({ title: "Shortcut ⌘K activated" })
```

## With description (expand on hover)

Toasts with a `description` expand on hover via the gooey SVG morphing animation:

```ts
sileo.success({
  title: "Payment accepted",
  description: "€49.00 charged to Visa ···· 4242",
})
```

## Action toast

```ts
sileo.action({
  title: "File deleted",
  description: "report-2024.pdf was moved to trash.",
  button: {
    title: "Undo",
    onClick: () => sileo.success({ title: "Restored" }),
  },
})
```

## Promise toast

Automatically transitions from loading → success or error:

```ts
sileo.promise(uploadFile(file), {
  loading: { title: "Uploading…" },
  success: (data) => ({
    title: "Upload complete",
    description: `${data.filename} · ${data.size}`,
  }),
  error: (err) => ({
    title: "Upload failed",
    description: err instanceof Error ? err.message : "Try again.",
  }),
})
```

## State transition (update by id)

Pass the same `id` to transition from one state to another:

```ts
const id = "payment"
sileo.loading({ title: "Processing…", id, duration: null })

setTimeout(() => {
  sileo.success({ title: "Payment accepted", id })
}, 2500)
```

## Persistent toast

Set `duration: null` to keep the toast until explicitly dismissed:

```ts
sileo.warning({
  title: "Unsaved changes",
  description: "Leave the page? Your edits won't be saved.",
  duration: null,
  id: "unsaved",
  button: {
    title: "Stay",
    onClick: () => sileo.dismiss("unsaved"),
  },
})
```

## Custom fill (dark surfaces)

```ts
sileo.success({
  title: "Dark fill toast",
  description: "Custom fill with style overrides.",
  fill: "#18181b",
  styles: {
    title: "my-dark-title",
    description: "my-dark-desc",
  },
})
```

```css
.my-dark-title { color: #fafaf2 !important; }
.my-dark-desc  { color: rgba(250, 250, 242, 0.55) !important; }
```
