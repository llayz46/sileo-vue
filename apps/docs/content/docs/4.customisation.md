# Customisation

## Theme

Set the `theme` prop on `<Toaster>`. Accepts `"light"`, `"dark"`, or `"system"` (default):

```vue
<Toaster theme="dark" />
```

Or drive it reactively:

```vue
<script setup>
const theme = ref<"light" | "dark" | "system">("system")
</script>

<template>
  <Toaster :theme="theme" />
</template>
```

## Position

Set the default position on `<Toaster>` or override per-toast:

```vue
<Toaster position="bottom-right" />
```

```ts
// Per-toast override
sileo.success({ title: "Saved", position: "bottom-center" })
```

Available positions: `top-left`, `top-center`, `top-right`, `bottom-left`, `bottom-center`, `bottom-right`.

## Duration

Default is 4000 ms. Override globally or per-toast:

```vue
<!-- Global default: 6 seconds -->
<Toaster :options="{ duration: 6000 }" />
```

```ts
// Per-toast
sileo.info({ title: "FYI", duration: 8000 })

// Persistent (never auto-dismisses)
sileo.loading({ title: "Processing…", duration: null })
```

## Custom fill

Override the background color of the SVG canvas:

```ts
sileo.success({
  title: "Custom background",
  fill: "#1e1b4b",  // any CSS color
})
```

## Roundness

Adjust the border radius (default: 16):

```ts
sileo.success({ title: "Sharper", roundness: 8 })
sileo.success({ title: "Pill-like", roundness: 24 })
```

## Style class overrides

Use `styles` to apply your own CSS classes to specific parts of the toast:

```ts
sileo.success({
  title: "Custom styled",
  description: "With custom classes.",
  styles: {
    title: "my-title-class",
    description: "my-desc-class",
    badge: "my-badge-class",
    button: "my-button-class",
  },
})
```

## Global defaults

Set default options for all toasts via the `options` prop on `<Toaster>`:

```vue
<Toaster
  :options="{
    duration: 5000,
    roundness: 12,
    position: 'bottom-right',
  }"
/>
```

## Offset

Control the distance from the viewport edge (default: 16px):

```vue
<Toaster :offset="24" />
```

## Autopilot

Automatically expand and collapse a toast with a description after a delay:

```ts
sileo.info({
  title: "Info",
  description: "This expands automatically.",
  autopilot: {
    expand: 500,    // expand after 500ms
    collapse: 3000, // collapse after 3000ms
  },
})
```
