# Getting Started

sileo-vue brings gooey SVG morphing and spring physics to Vue 3 — a faithful port of [sileo](https://sileo.aaryan.design) by aaryan.design.

## Installation

```bash
pnpm add sileo-vue
# or
npm install sileo-vue
# or
yarn add sileo-vue
```

## Setup

Add `<Toaster>` once in your root component (e.g. `App.vue`) and import the stylesheet:

```vue
<script setup>
import { Toaster } from "sileo-vue"
import "sileo-vue/styles.css"
</script>

<template>
  <Toaster position="top-right" />
  <RouterView />
</template>
```

## Trigger a toast

Call `sileo.*` from anywhere — no context, no inject needed:

```ts
import { sileo } from "sileo-vue"

sileo.success({ title: "Saved!" })
sileo.error({ title: "Failed", description: "Something went wrong." })
sileo.loading({ title: "Uploading…" })
```

## Nuxt

In a Nuxt app, add the `<Toaster>` in `app.vue`:

```vue
<script setup>
import { Toaster } from "sileo-vue"
import "sileo-vue/styles.css"
</script>

<template>
  <Toaster position="top-right" />
  <NuxtPage />
</template>
```

`sileo.*` calls work client-side from any component or composable.

## Laravel + Inertia

In a Laravel application using [Inertia.js](https://inertiajs.com) with Vue 3, mount `<Toaster>` in your root layout component.

Install the package:

```bash
npm install sileo-vue
```

In `resources/js/Layouts/AppLayout.vue` (or your shared layout):

```vue
<script setup>
import { Toaster } from "sileo-vue"
import "sileo-vue/styles.css"
</script>

<template>
  <Toaster position="top-right" />
  <slot />
</template>
```

Then call `sileo.*` from any page component:

```vue
<!-- resources/js/Pages/Profile/Edit.vue -->
<script setup>
import { useForm } from "@inertiajs/vue3"
import { sileo } from "sileo-vue"

const form = useForm({ name: "" })

function submit() {
  form.patch(route("profile.update"), {
    onSuccess: () => sileo.success({ title: "Profile updated" }),
    onError: () => sileo.error({ title: "Something went wrong" }),
  })
}
</script>
```

For form submissions that involve file uploads or long operations, use `sileo.promise()`:

```ts
import { router } from "@inertiajs/vue3"
import { sileo } from "sileo-vue"

function uploadAvatar(file: File) {
  const data = new FormData()
  data.append("avatar", file)

  sileo.promise(
    new Promise<void>((resolve, reject) => {
      router.post(route("profile.avatar"), data, {
        onSuccess: () => resolve(),
        onError: () => reject(new Error("Upload failed")),
      })
    }),
    {
      loading: { title: "Uploading avatar…" },
      success: () => ({ title: "Avatar updated" }),
      error: () => ({ title: "Upload failed", description: "Please try again." }),
    }
  )
}
```
