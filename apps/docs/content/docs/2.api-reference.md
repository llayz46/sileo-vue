# API Reference

## `sileo` methods

All methods accept a `SileoOptions` object and return the toast `id`.

### `sileo.success(options)`

```ts
sileo.success({ title: "Changes saved" })
sileo.success({ title: "Done", description: "All 3 files uploaded." })
```

### `sileo.error(options)`

```ts
sileo.error({ title: "Upload failed", description: "Max 10 MB." })
```

### `sileo.warning(options)`

```ts
sileo.warning({ title: "Storage at 90%" })
```

### `sileo.info(options)`

```ts
sileo.info({ title: "v2.4 available", description: "See changelog." })
```

### `sileo.loading(options)`

Stays visible until updated or dismissed manually.

```ts
const id = sileo.loading({ title: "Uploading…", duration: null })
```

### `sileo.action(options)`

Shows a toast with a button.

```ts
sileo.action({
  title: "File deleted",
  button: { title: "Undo", onClick: () => restore() },
})
```

### `sileo.show(options)`

Generic toast without an icon.

```ts
sileo.show({ title: "Shortcut ⌘K activated" })
```

### `sileo.promise(promise, handlers)`

Transitions from loading → success or error automatically.

```ts
sileo.promise(fetch("/api/save"), {
  loading: { title: "Saving…" },
  success: (data) => ({ title: "Saved!", description: data.message }),
  error: (err) => ({ title: "Error", description: err.message }),
})
```

### `sileo.dismiss(id?)`

Dismiss a specific toast by id, or all toasts if no id provided.

```ts
sileo.dismiss("my-toast-id")
```

### `sileo.clear()`

Dismiss all toasts immediately.

```ts
sileo.clear()
```

## `<Toaster>` props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `SileoPosition` | `"top-right"` | Default position for toasts |
| `theme` | `"light" \| "dark" \| "system"` | `"system"` | Color theme |
| `offset` | `number` | `16` | Offset from viewport edge (px) |
| `options` | `Partial<SileoOptions>` | — | Global default options for all toasts |
| `debug` | `boolean` | `false` | Show debug outlines on SVG layers |

---

## `SileoOptions`

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `id` | `string` | auto | Unique id (used for updates) |
| `title` | `Renderable` | — | Toast title |
| `description` | `Renderable` | — | Expandable description (hover to reveal) |
| `position` | `SileoPosition` | `"top-right"` | Per-toast position override |
| `duration` | `number \| null` | `4000` | Auto-dismiss delay in ms. `null` = persistent |
| `fill` | `string` | — | Custom background color |
| `roundness` | `number` | `16` | Border radius in px |
| `icon` | `Renderable` | — | Custom icon (replaces default state icon) |
| `button` | `SileoButton` | — | Action button `{ title, onClick }` |
| `styles` | `SileoStyles` | — | CSS class overrides per slot |
| `autopilot` | `SileoAutopilot` | — | Auto expand/collapse timing `{ expand, collapse }` |

## `useToast()`

Composable to reactively access the current toast list:

```ts
import { useToast } from "sileo-vue"

const { toasts } = useToast()
// toasts: Ref<SileoItem[]>
```

## Types

```ts
type SileoPosition =
  | "top-left" | "top-center" | "top-right"
  | "bottom-left" | "bottom-center" | "bottom-right"

type SileoState =
  | "success" | "error" | "warning" | "info"
  | "loading" | "action" | "show"

type Renderable = string | VNode | Component | null | undefined

interface SileoButton {
  title: string
  onClick: () => void
}

interface SileoStyles {
  title?: string
  description?: string
  badge?: string
  button?: string
}
```
